# Evidence-Grounded Drafting Skill

## Purpose

Generate draft grant answers using only uploaded nonprofit evidence.

## When to Use

Use this skill when creating or reviewing application-drafting logic.

## Rules

- Every factual claim should be supported by a citation.
- If evidence is missing, say so.
- Do not invent impact metrics.
- Do not invent budget numbers.
- Keep the tone professional and grant-appropriate.
- Make the answer editable by the user.

## Draft Answer Structure

For each question:

- question
- draft_answer
- supporting_citations
- missing_evidence
- confidence
- improvement_suggestions
