# PDF Packet Generation Skill

## Purpose

Create a polished grant readiness report PDF.

## Required Sections

- Cover page
- Organization summary
- Grant summary
- Eligibility score
- Readiness score
- Requirement checklist
- Missing documents
- Risk flags
- Draft answers
- Evidence citations
- Recommended next steps

## Rules

- Keep the report readable.
- Use clear section headings.
- Include citations.
- Do not include raw internal prompts.
